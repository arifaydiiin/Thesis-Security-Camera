# thesis

Bu çalışmada esp32 kamera modülü ile mobil uygulamadan 7/24 izlenebilen,hareket algılandığında bildirim alan güvenlik kamera sistemi geliştirilmiştir.

Gerekli Malzemeler;
- Esp32-Cam Kamera modülü
- PIR hareket sensörü
- 10k ohm Direnç
- 1k ohm Direnç
- 220 ohm Direnç
- BC547 Transistor
- Usb Kablosu
- Led
